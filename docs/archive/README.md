# SaaS App

## Project Overview
This SaaS app provides core functionality for users to manage their profiles, subscriptions, and interact with a rich dashboard.

## Prerequisites
- Node.js v16 or higher
- npm (Node package manager)

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app locally:
   ```bash
   npm start
   ```
4. Run tests:
   ```bash
   npm test
   ```
